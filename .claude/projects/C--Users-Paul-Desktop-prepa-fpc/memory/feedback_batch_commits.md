Regrouper les commits et push toutes les 3-4 modifications, pas à chaque petit changement.

**Why:** L'utilisateur trouve trop verbeux de push à chaque modification mineure. Ça pollue l'historique git.

**How to apply:** Accumuler 3-4 edits avant de commit+push, sauf si l'utilisateur demande explicitement de push.
