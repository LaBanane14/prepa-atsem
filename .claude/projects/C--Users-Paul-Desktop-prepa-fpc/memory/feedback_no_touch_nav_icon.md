Ne jamais modifier l'icône stéthoscope SVG dans les barres de navigation / headers des pages.

**Why:** L'utilisateur considère que l'icône actuelle (SVG 24x24 stroke-based) est correcte dans l'en-tête. Les modifications précédentes (remplacement par le SVG 100x100 du favicon) ont dégradé le rendu.

**How to apply:** Quand on travaille sur le favicon ou d'autres icônes, ne pas toucher aux SVG inline dans les nav/headers des pages (page.js, qcm, blog, tarifs, calculs-doses, dashboard, oral, etc.).
